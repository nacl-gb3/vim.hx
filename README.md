# vim.hx

Vim bindings for helix, using Steel.

## Installation

```shell
forge pkg install --git https://github.com/mattwparas/vim.hx.git
````

## Usage

```scheme
(require "vim-hx/init.scm")

;; Add this to your init.scm
(set-vim-keybindings!)
```
